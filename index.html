3D Flip-Dot Display - Help

Keyboard Shortcuts

AToggle GUI parameter panel
DToggle UI (buttons & image inputs)
MCycle Mouse Trail mode (Off / Straight / Chevron)
FFit camera to fill window with dots
?Toggle this help

Animation Buttons

RippleCenter-outward ripple flip
Down / UpRow-by-row flip (top-down / bottom-up)
Right / LeftColumn-by-column flip
WindRandom direction sweep with turbulence
Natural WindMulti-gust organic wind with irregular wavefront
Click RippleToggle mode: click anywhere to ripple from that point
Mouse TrailToggle drag-to-flip: Straight (column sweep) or Chevron (V-wave)

Control Buttons

ResetReset all dots to front face, cancel animations
ShapeToggle dot shape: Round (spheroid) / Square (box)
GapToggle gap between dots: On (spaced) / Off (tight)
GlossyToggle surface finish: Matte / Glossy
BGCycle background color: White / Black / Gray
True ColorToggle unlit mode: show original image colors without lighting
FitAuto-fit camera so dots fill the window

GUI Parameters (Panel)

FPSCurrent frames per second
ColumnsNumber of dots horizontally (4-96)
RowsNumber of dots vertically (4-54)
SpacingDistance between dot centers
SquashDot thickness (0.1=flat, 1.0=sphere)
RoughnessSurface roughness (0=mirror, 1=matte)
MetalnessMetallic appearance (keep low to preserve colors)
Flip DurationTime for a single dot flip (seconds)
Stagger DelayDelay between dots in Ripple/Click Ripple
Line DelayDelay between lines in directional animations
Trail RadiusMouse Trail: cursor influence radius
Trail GradientChevron mode: V-wave spread (0=straight)
